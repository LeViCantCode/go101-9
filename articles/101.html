Go 101

Go 101 is a book focusing on Go syntax/semantics and all kinds of details (Go 1.13 ready). This book also tries to help gophers gain a deep and thorough understanding of Go. It is expected that this book is helpful for both beginner and experienced Go programmers.

  • The go101.org website is hosted on App Engine. Surely, it is built with Go.
  • 中文版在这里。
  • The Go 101 project is hosted on both Gitlab and Github. You can clone the code, then run go run . from the project path to start a local go101 server for faster browsing.
  • Go 101's official twitter account is @go100and1. Some Go details, facts, and quizzes were (and will be) posted there from time to time.
